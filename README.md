# 3D Scene Project – Computational Graphics Portfolio

## Project Overview
This project demonstrates my ability to design and develop a 3D scene using OpenGL. The scene includes a constructed table made from multiple 3D shapes, along with applied textures and transformations. The project was developed using Visual Studio and includes a runnable executable located in the Debug folder.

---

## Repository Contents

- 7-1_FinalProjectMilestones.sln (Visual Studio solution file)
- Source files (.cpp and .h)
- Debug/7-1_FinalProjectMilestones.exe (runnable application)
- Project configuration files

## How do I approach designing software?

Through this project, I developed skills in designing 3D environments by breaking a scene into smaller components such as shapes, textures, and transformations. I focused on structuring the table using individual objects (tabletop and legs) and ensuring proper alignment.

My design process included:
- Planning the layout of the scene
- Building objects step-by-step
- Adjusting scale, position, and textures

This approach can be applied in future projects by breaking down complex systems into smaller, manageable components and improving them iteratively.

---

## How do I approach developing programs?

While developing this project, I used strategies such as incremental testing and debugging. I frequently ran the program to check object placement, texture application, and camera positioning.

Iteration was a key part of development:
- Initial versions had alignment and texture issues
- I refined object positioning and scaling through multiple tests
- I corrected errors step-by-step until the scene rendered correctly

My development approach improved by becoming more structured, focusing on testing small changes instead of making large changes all at once.
